## Struktur dasar HTML

Struktur dasar HTML terdiri dari 5 element, yaitu : 
```<!DOCTYPE html>``` <br>
```<html>``` <br>
```<head>``` <br>
```<title>``` <br>
```<body>``` <br>

Berikut ini contoh penulisannya secara lengkap.

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <title>SaCode</title>
</head>
<body>

    <p>Hello SaCode!</p>
    
</body>
</html>
```

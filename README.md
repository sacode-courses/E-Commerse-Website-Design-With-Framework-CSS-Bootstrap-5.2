## Struktur dasar HTML

Struktur dasar HTML terdiri dari 5 element, yaitu : 
```<!DOCTYPE html>```
```<html>```
```<head>```
```<title>```
```<body>```

Berikut ini contoh penulisannya secara lengkap.

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <title>SaCode</title>
</head>
<body>

    <p>Hello SaCode!</p>
    
</body>
</html>
```
